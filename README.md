# Процентовка и программы

PWA для пауэрлифтинга. Пять разделов:

- **Калькулятор** — вводишь 1ПМ (присед, жим лёжа, становая) → веса по **4 неделям цикла** сразу (Н1–Н2 средние, Н3 тяжёлая, Н4 лёгкая), плюс сумма троеборья и числовая цель цикла (105 %). Веса до 0,1 кг. **105 % (проходка) в таблицах не показывается** — это соревнования, не тренировка.
- **Программы** — шаблоны (наш 4-нед. цикл, 5/3/1, Шейко, Куб) и конструктор своих программ. _(в разработке)_
- **Журнал** — план (просмотр программы) и факт (дневник: повторы, вес, качество 1–5, ошибки, заметки). _(в разработке)_
- **Подсобки** — каталог по категориям и подбор подсобок под типовую ошибку. _(в разработке)_
- **Профиль** — аватар, ФИО, возраст; вес тела с историей; тайминг тренировки; роль Тренер/Атлет.

Полный план доработки — в [`TZ.md`](TZ.md). Передача проекта — в [`HANDOFF.md`](HANDOFF.md).

🌐 Сайт: https://bablakov.github.io/calculator-vesov/

## Структура

| Путь | Назначение |
|---|---|
| `index.html` | Каркас: шапка, навигация (5 разделов), контейнеры |
| `css/styles.css` | Все стили |
| `js/app.js` | Точка входа |
| `js/nav.js` | Навигация по hash (`#calc`/`#programs`/`#journal`/`#accessories`/`#profile`) |
| `js/config.js` | Предметная конфигурация: `LIFTS`, `WEEKS` (матрица), `SPLIT`, `QUALITY` |
| `js/calc.js` | Калькулятор: 4 таблицы недель |
| `js/profile.js` | Профиль (данные, вес тела, тайминг, роль) |
| `js/cycle.js` | Движок программы из 1ПМ (для Журнала) |
| `js/content.js` | Справочники: ошибки→инструкции, обучение (для Журнала/Подсобок) |
| `js/util.js`, `js/store.js` | Утилиты и обёртка `localStorage` |
| `js/pwa.js` | Установка, «Поделиться», service worker |
| `sw.js` | Service worker — офлайн-кэш (`CACHE`) |
| `tests/` | Тесты (node ESM): `npm test` |
| `manifest.webmanifest`, `icon.*`, `qr.svg` | PWA-метаданные, иконки, QR |
| `tools/gen-*.mjs` | Генераторы иконок и QR (dev) |

## Что менять под себя

Предметная конфигурация — в `js/config.js`: `LIFTS` (упражнения и цвета), `WEEKS` (матрица недель: проценты и повторы), `SPLIT` (дни сплита), `TARGET_PCT`.

## Локальный запуск

```bash
python -m http.server 8000   # → http://localhost:8000
```
SW и установка требуют http(s) или localhost. После правок — **Ctrl+Shift+R**.

## Тесты

```bash
npm test     # node tests/run.mjs — проверки ядра (матрица, округление) и логики профиля
```

## Публикация (GitHub Pages)

Деплой = `git push` в `main` (Pages: `main` / `root`).
**При любом изменении файлов поднимай версию `CACHE` в `sw.js`** (`pct-v10` → `pct-v11` …).

## Перегенерация иконок / QR

```bash
node tools/gen-icons.mjs
npm install && npm run qr     # или: node tools/gen-qr.mjs "https://новый-адрес/"
```
