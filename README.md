# Процентовка и программы

PWA для пауэрлифтинга. Семь разделов:

- **Калькулятор** — вводишь 1ПМ (присед, жим лёжа, становая) → веса по **4 неделям цикла** сразу (Н1–Н2 средние, Н3 тяжёлая, Н4 лёгкая), плюс сумма троеборья и числовая цель цикла (105 %). Веса до 0,1 кг. **105 % (проходка) в таблицах не показывается** — это соревнования, не тренировка.
- **Программы** — мастер создания в 3 шага (уровень → дисциплина → упор) даёт готовую программу от 1ПМ; конструктор правит недели → дни → подходы (%≤100) и подсобки к движениям.
- **Журнал** — план и факт по **неделям и дням**; автопрогрессия (какая тренировка сейчас); вес тела и тайминг — в самой тренировке; дневник (вес/«на снарядах»/повторы/качество/ошибки/заметки) + история.
- **Отчёты** — статистика по тренировкам: число, средняя/мин/макс длительность, частота, объём, качество, тоннаж по движениям.
- **Подсобки** — каталог по категориям и подбор подсобок под типовую ошибку.
- **Профиль** — аватар, ФИО, дата рождения (возраст считается); вес тела (текущий — из последней тренировки) с историей.
- **Админ** — правка контента: тексты ошибок, связи «проблема→подсобки», каталог подсобок (переопределяют дефолты, есть сброс).

Полный план доработки — в [`TZ.md`](TZ.md). Передача проекта — в [`HANDOFF.md`](HANDOFF.md).

## Что менять под себя

Предметная конфигурация — в `js/config.js`: `LIFTS` (упражнения и цвета), `WEEKS` (матрица недель: проценты и повторы), `SPLIT` (дни сплита), `TARGET_PCT`.

## Локальный запуск

```bash
python -m http.server 8000   # → http://localhost:8000
```
SW и установка требуют http(s) или localhost. После правок — **Ctrl+Shift+R**.

## Тесты

```bash
npm test     # node tests/run.mjs — матрица/округление, профиль, программы, журнал, подсобки
```

## Публикация (GitHub Pages)

Деплой = `git push` в `main` (Pages: `main` / `root`).
**При любом изменении файлов поднимай версию `CACHE` в `sw.js`** (`pct-v12` → `pct-v13` …).

## Перегенерация иконок / QR

```bash
node tools/gen-icons.mjs
npm install && npm run qr     # или: node tools/gen-qr.mjs "https://новый-адрес/"
```
